# Tank Game

A 3-player multiplayer tank battle game built with vanilla JavaScript and p5.js.

## What It Is

A real-time tank combat game where three players control tanks using different input methods (WASD keys, arrow keys, and mouse) and battle each other on a destructible arena. Players can pick from different tank skins before the game starts, fire projectiles, collect power-ups, and compete to be the last tank standing.

## Tech Stack

- **p5.js** - Canvas-based graphics and interaction
- **Vanilla JavaScript** - Game logic and physics
- **HTML5 Canvas** - Rendering

## Features

- **Three-player gameplay** with distinct control schemes:
  - Player 1: WASD to move, Spacebar to shoot
  - Player 2: Arrow keys to move, Enter to shoot
  - Player 3: Mouse to aim and click to shoot
- **Tank customization** - Select from multiple tank skins before playing
- **Power-ups** - Speed boosts, shields, and rapid fire abilities
- **Physics simulation** - Projectile trajectories and collision detection
- **Dynamic arena** - Destructible walls and environmental gameplay
- **Real-time HUD** - Score tracking and FPS display

## Getting Started

1. Open `index.html` in a web browser
2. Select a tank skin for each player
3. Use your assigned controls to move and shoot
4. Last tank standing wins!

The game is fully self-contained and requires no build process or dependencies beyond p5.js (loaded via CDN).
